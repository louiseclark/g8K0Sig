Beam Asymmetry - comparison of methods

Comparison with toy data

  • Toy events are created starting from phase space simulation data
  • Observable values are set equal to those extracted for the real data
  • Simulated events are then accepted/rejected based on the corresponding distribution
  • 5 sets of toy data are generated, with number of events in each set approximately equal to the integrated signal for that bin
